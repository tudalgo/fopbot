DrawingRegistryBuilder

public class DrawingRegistryBuilder(source)

A builder for creating instances of DrawingRegistry.

This builder allows you to add drawables to the registry, which are associated with the FieldEntity classes they belong to. You can add individual drawables or a collection of them, and then build the DrawingRegistry using the build method, which requires a custom comparator for sorting the field entities.
